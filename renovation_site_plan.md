# リノベーション事業HP 構成案

## 1. サイトコンセプト

白を基調に、自然素材・グリーン・余白を活かしたナチュラルで洗練されたリノベーション事業サイト。
高級すぎず、安っぽくもない。「暮らしを整える」「余白のある住まい」「自然体で心地よい家」を伝える。

## 2. デザイン方向性

- ベースカラー：ホワイト、アイボリー、淡いグレージュ
- アクセントカラー：オリーブグリーン、セージグリーン、木目ブラウン
- フォント：明朝系、細めのセリフ体、上品な日本語フォント
- 写真：木材、観葉植物、自然光、白壁、リビング、キッチン、窓辺
- 余白：広め
- 印象：静か、上品、自然、丁寧、暮らし寄り

### 4-3. コンセプト

見出し：

> 住まいを、暮らしに合わせて整える。

本文案：

古くなった家を新しくするだけではなく、家族の時間、光の入り方、収納、動線、素材の手触りまで見直し、日々の暮らしが自然と整う住まいを提案します。

### 4-4. お悩み提示

見出し：

> こんなお悩みはありませんか？

項目：

- 中古住宅を購入して、自分好みにリノベーションしたい
- 古い間取りを、今の暮らしに合う形に変えたい
- 自然素材を使った落ち着いた空間にしたい
- キッチンや水回りをもっと使いやすくしたい
- 予算内でどこまでできるのか知りたい
- 工事の流れや費用感がわからず不安

### 4-5. サービス内容

カード形式で表示する。

1. マンションリノベーション
   - 間取り変更、収納改善、内装デザインまで一括対応。

2. 戸建てリノベーション
   - 既存の良さを活かしながら、断熱・耐震・動線を見直す。

3. 水回りリフォーム
   - キッチン、浴室、洗面、トイレを使いやすく美しく。

4. 自然素材リノベーション
   - 無垢材、珪藻土、漆喰などを使った心地よい空間づくり。

5. 中古物件購入＋リノベ相談
   - 物件探しの段階から、リノベ向きかどうかを確認。

6. 店舗・小規模オフィス改装
   - カフェ、美容室、サロン、小規模店舗の内装設計。

### 4-7. 選ばれる理由

見出し：

> 私たちが大切にしていること

項目：

- 自然素材を活かしたシンプルなデザイン
- 見積もり内容をわかりやすく説明
- 設計から施工まで一貫対応
- 生活動線・収納計画まで丁寧に提案
- 工事後のアフターサポート
- 必要に応じて保証・保険制度も案内

### 4-8. 費用の目安

| 内容 | 価格目安 |
|---|---:|
| 水回りリフォーム | 80万円〜 |
| 部分リノベーション | 150万円〜 |
| マンション全面リノベーション | 600万円〜 |
| 戸建て全面リノベーション | 900万円〜 |

補足文：

建物の状態、工事範囲、使用する素材によって費用は変動します。まずは現地調査とヒアリングを行い、無理のない予算計画をご提案します。

## 6. 制作時の注意点

- 施工事例の写真を最優先で魅力的に見せる
- 価格は断定ではなく「目安」として表示する
- 見積もり、契約、保証、アフターサービスの説明を入れて安心感を出す
- テキストは売り込みすぎず、丁寧で落ち着いた言葉にする
- 余白、行間、写真のトーンを揃える
